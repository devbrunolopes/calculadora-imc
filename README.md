# calculadora-imc

Esse é um aplicativo para calcular o IMC (índice de massa corporal). Basicamente você coloca o seu peso e a sua altura e ele faz todo o cálculo.

O QUE APRENDI NESSE PROJETO:

- Criar múltiplas telas
- MVC
- Class vc Struct
- Optionals
